# Adey Wallet App 💳

A modern, animated mobile wallet application built with Flutter featuring a beautiful UI, smooth animations, and comprehensive financial management features.

## ✨ Features

### 🔐 Authentication
- Elegant login screen with animated logo
- Email and password authentication
- Password visibility toggle
- Responsive keyboard handling

### 🏠 Dashboard
- Real-time balance display with gradient card
- Premium customer status badge
- Animated wallet icon illustration
- Quick action buttons (Send, Receive, Pay Bills, More)
- Recent transaction history
- Smooth fade and slide animations

### 💳 Cards Management
- Multiple card support (Visa, Mastercard)
- Beautiful card designs with gradients
- Card settings (Freeze, Limits, Security)
- Hero animations for smooth transitions

### 📊 Transactions
- Complete transaction history
- Categorized transactions with icons
- Debit/Credit color coding
- Transaction details (date, time, amount)

### 👤 Profile
- User information display
- Premium membership status
- Settings and preferences
- Animated avatar with gradient
- Account management options

## 🎨 UI/UX Highlights

- **Smooth Animations**: Fade, slide, scale, and rotation animations throughout
- **Modern Design**: Material 3 design with custom gradients and shadows
- **Responsive Layout**: Adapts to different screen sizes
- **Bottom Navigation**: Easy navigation between main sections
- **Color Coded**: Intuitive color system (green for credit, red for debit)
- **Icons & Illustrations**: Rich iconography for better visual communication

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (3.0 or higher)
- Dart SDK (3.0 or higher)
- Android Studio / VS Code
- Android Emulator or iOS Simulator
